# 🦄 Пинки Пай Бот

Весёлый и дружелюбный бот, созданный по образу Пинки Пай из мультсериала "Дружба — это чудо".

> **Статус проекта:** ✅ **Активная разработка**
>
> Проект изначально разрабатывался для мессенджера **MAX** с использованием российских ИИ-сервисов (GigaChat, YandexGPT).
>
> **Причина смены платформы:** На данный момент создание ботов в MAX доступно только для юридических лиц, ИП и самозанятых (резидентов РФ). Физические лица не могут пройти верификацию на платформе.
>
> **Текущее состояние:** Бот успешно портирован на **Telegram** с переходом на **OpenAI GPT-4-turbo**. Все функции сохранены и расширены.
>
> 👨‍💻 *Автор: MADAO81*

## 📖 О проекте

Пинки Пай — это интерактивный бот, который:

- 🎉 Общается в характере персонажа: весело, энергично и дружелюбно
- 🧁 Делится рецептами выпечки и сладостей (база данных SQLite, 113+ рецептов)
- 🎶 Рассказывает шутки и поёт песенки
- 🌦️ Меняет настроение в зависимости от погоды в Ворсино (Боровский район)
- 🌍 Показывает погоду в любом городе мира (с поддержкой падежей и предлога «во»)
- 📝 Комментирует сообщения в группах при упоминании или с вероятностью 20%
- 🎙️ Распознаёт голосовые сообщения (Whisper) с вероятностью 20%
- 🖼️ Анализирует изображения (GPT-4 Vision) с вероятностью 20%
- 🧁 Ежедневная рассылка рецептов в 12:00 по подписке
- ⏰ Работает в установленные часы (9:00–20:00)
- 💾 Хранит историю диалогов (30 дней) с возможностью очистки
- 🔒 Прозрачное уведомление о хранении данных и команда `/cleardata`

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/your-username/pinkie-pie-bot.git
cd pinkie-pie-bot
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # для Linux/Mac
# или
venv\Scripts\activate     # для Windows
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения

Скопируйте файл `.env.example` в `.env` и заполните своими ключами:

```env
# Telegram (обязательно)
TELEGRAM_TOKEN=ваш_токен_бота_от_BotFather

# OpenAI (обязательно)
OPENAI_API_KEY=ваш_api_ключ_openai
OPENAI_MODEL=gpt-4-turbo

# Координаты Ворсино (Боровский район)
DEFAULT_LAT=55.0965
DEFAULT_LON=36.6355

# Настройки бота (опционально)
WORK_START_HOUR=9
WORK_END_HOUR=20
SAD_PROBABILITY=0.2
CONTEXT_EXPIRE_DAYS=30
RECIPE_SEND_TIME=12:00

# Режим отладки
DEBUG_MODE=false
```

### 5. Создайте базу данных рецептов

```bash
python bot/scripts/create_recipe_db.py
python bot/scripts/fill_recipes_db.py
```

### 6. Запустите бота

```bash
python run.py
```

## 🌤️ Погода

### По умолчанию (Ворсино)
- Координаты: `55.0965, 36.6355`
- Настроение Пинки Пай зависит от погоды в Ворсино
- Вероятность грусти в плохую погоду: **20%**

### В любом городе
- Поддерживаются запросы на русском и английском
- Понимает падежи: `в Москве`, `в Лондоне`, `в Берлине`
- Понимает предлог «во»: `во Владивостоке`
- Автоматическая нормализация названий через pymorphy2
- Примеры:
  - `погода в Москве`
  - `погода в Лондоне`
  - `температура в Париже`
  - `что за окном в Берлине?`
  - `погода во Владивостоке`

## 🧁 Ежедневные рецепты

- Каждый день в **12:00** бот отправляет случайный рецепт выпечки
- Чтобы получать рецепты, подпишитесь командой `/subscribe`
- Отписаться можно командой `/unsubscribe`
- Рецепты берутся из базы данных SQLite (113+ рецептов)

## 📝 Особенности поведения

### 🎯 Реакция на сообщения в группах

| Ситуация | Поведение |
|----------|-----------|
| Упоминание бота (`@username`) | ✅ Всегда отвечает |
| Ответ на сообщение бота | ✅ Всегда отвечает |
| Обычное сообщение (без упоминания) | ✅ С вероятностью **20%** |
| Личное сообщение | ✅ Всегда отвечает |

### 🎯 Реакция на медиа

| Тип | Вероятность ответа |
|-----|-------------------|
| Фото | **20%** |
| Голосовые | **20%** |

### ⏰ Рабочее время

- **Понедельник — Воскресенье:** 9:00 — 20:00
- **Вне рабочего времени:** игнорирует сообщения в группах, в личке отвечает: *"Я отдыхаю, приходите завтра!"*

### 🧁 Книга рецептов

- **База данных SQLite** с 113+ рецептами выпечки
- Резервные рецепты в коде (на случай ошибок)
- Категории: торты, пирожные, печенье, кексы, пироги, десерты, другое
- Ежедневная рассылка в 12:00 по подписке (`/subscribe`)

## 🔒 Конфиденциальность и данные

- Бот сохраняет историю диалогов только для поддержания контекста беседы
- Данные хранятся локально на сервере в SQLite
- Никакие данные не передаются третьим лицам
- Пользователь может удалить свою историю командой `/cleardata`
- Предупреждение о хранении данных отображается в `/start` и `/help`

## 🚀 Деплой на сервер

### Управление через systemd

```bash
# Запустить бота
systemctl start pinkie-bot

# Остановить бота
systemctl stop pinkie-bot

# Перезапустить бота
systemctl restart pinkie-bot

# Проверить статус
systemctl status pinkie-bot

# Просмотр логов
journalctl -u pinkie-bot -f
```

### Обновление через Git

```bash
systemctl stop pinkie-bot
git pull origin master
systemctl start pinkie-bot
```

### Мониторинг и автоперезапуск

На сервере настроен cron-скрипт, который каждые 5 минут проверяет доступность бота и автоматически перезапускает его при зависании:

```bash
# Просмотр логов сторожевого скрипта
cat /var/log/bot_watchdog.log

# Ручной запуск проверки
/root/check_bot.sh
```
